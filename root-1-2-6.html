    For \(k\in\mathbb{N}\) and a nonempty set \(A\text{,}\) a \(k\)-permutation of \(A\) is an injective function \(w:[k]\to A\text{.}\) The set of all \(k\)-permutations of \(A\) is denoted by \({S_{A,k}}\text{.}\) If \(A\) happens to be the set \([n]\text{,}\) we use the notation \({S_{n,k}}\text{.}\) And if \(n=k\text{,}\) we write \({S_n:=S_{n,n}}\) and refer to each \(n\)-permutation in \(S_n\) as a permutation. Let \({P(n,k):=|S_{n,k}|}\text{.}\) By convention, we set \({P(n,0):=1}\text{,}\) including the case when \(n=0\text{.}\)

    We can think of a \(k\)-permutation as a linearly-ordered arrangement (i.e., string) of \(k\) of \(n\) objects. That is, we can denote a \(k\)-permutation as a string \(w=w(1)w(2)\cdots w(k)\text{,}\) where each \(w(i)\in [n]\) and \(w(i)\neq w(j)\) for \(i\neq j\text{.}\) For example, if \(n=6\) and \(k=4\text{,}\) then the string \(3624\) represents the \(4\)-permutation \(w:[4]\to [6]\) given by

    Consider a set of \(n\) objects that are not necessarily distinct, with \(p\) different types objects and \(n_i\) objects of type \(i\) (for \(i=1,2,\ldots, p\)), so that \(n=n_1+\cdots +n_p\text{.}\) An ordered arrangement of these \(n\) objects is called a generalized permutation and the number of such arrangements is denoted by \({P(n;n_1,\ldots,n_p)}\text{.}\) For example, the number of words we can make out of the letters of POPPY is \(P(5;3,1,1)\text{.}\) The following theorem follows immediately from the Division Principle.

    Theorem 3.10.

    For \(n,n_1,\ldots,n_p\in \mathbb{N}\) such that \(n=n_1+\cdots +n_p\text{,}\) we have
    \begin{equation*} {P(n;n_1,\ldots,n_p)=\frac{n!}{n_1!\cdots n_p!}.} \end{equation*}
